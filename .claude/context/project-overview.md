# 项目概述

## 项目名称
AI白板 ([aibaiban.com](https://aibaiban.com/))

## 项目定位
一个基于 AI 的在线协作白板工具，旨在提供智能化的视觉协作体验。

## 核心功能（进行中）
- [x] 在线白板基础功能（基于 Excalidraw）
- [x] 丰富的素材库（Excalidraw 官方素材库）
- [x] 主题切换（DaisyUI 32 个预设主题）
- [x] AI 聊天面板 UI（Mock 版本）
- [ ] AI 辅助绘图
- [ ] AI 智能识别和建议
- [ ] 实时协作
- [ ] 智能内容识别
- [ ] ...

## 技术基础
- **白板框架**: Excalidraw 0.18.0
- **前端**: React 19.2.3 + TypeScript 5.9.3 + Vite 7.3.1
- **样式**: Tailwind CSS v4 + DaisyUI 5
- **路由**: React Router 7.12.0
- **后端**: Node.js（待实现）
- **架构**: Monorepo (pnpm 10.28.0 + Turborepo 2.7.4)
